3. Route Map
Path Screen Guard
/ LandingScreen
/sign-in SignInScreen redirect if signed in
/sign-up SignUpScreen redirect if signed in
/feed HomeNavigator (page 0) AUTH
/search HomeNavigator (page 1)
/market HomeNavigator (page 2)
/settings HomeNavigator (page 3)
/profile/:userId ProfileScreen
/chats ConversationListScreen AUTH
/profile-settings ProfileSettingScreen
/notification-settings NotificationSettingScreen
